# 模板图片 (Template Images)

此目录用于存放玩家编号的模板图片，作为 OCR 失败时的 fallback 识别方案。

## 使用方法

1. 打开鹅鸭杀游戏，进入发言阶段
2. 截图当前画面
3. 从截图中裁剪出每个玩家编号数字的小图片
4. 保存为 **灰度 PNG**，命名格式为 `XX.png`（两位数字，如 `01.png`, `02.png`）
5. 图片应只包含数字部分，尺寸尽量一致

## 需要的文件

当前代码中预设了以下编号（可在 `extract_speaker_num.py` 中扩展）：

```
01.png  02.png  06.png  10.png  11.png  12.png  13.png
```

## 快速生成方法

1. 启动服务后调用 `POST /api/debug-ocr?enabled=true`
2. 进入游戏发言阶段，让监控运行几帧
3. 查看 `_debug_crops/` 目录中保存的 `full_*.png`（完整截图）和 `roi_*.png`（裁剪区域）
4. 从完整截图中手动裁剪数字区域作为模板
